# VTEX IO Grafana Datasource

This is the TechDocs entry point for the VTEX IO Grafana Datasource plugin.

The plugin exposes VTEX IO Observability logs and metrics as a Grafana data source. It is a frontend-only Grafana plugin: queries flow from the Grafana React UI through the Grafana data-source proxy to the VTEX Observability API.
